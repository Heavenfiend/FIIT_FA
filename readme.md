## ⚙️ Как работать с репозиторием

Для выполнения лабораторных необходимо вести разработку с использованием GitHub.

1. Сделайте `fork` (кнопочка сверху справа).
2. У вас появится личная копия репозитория, внутри которого вы и работаете.
3. Склонируйте его на свой компьютер и можете приступать к работе.

> **Да прибудет с GEMINI сила.** 🚂
